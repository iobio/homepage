gene.iobio
Gene.iobio provides variant prioritization and interrogation for research investigation and discovery in real-time using informative and engaging visualizations.

Workflow:

  • Search on a gene or a phenotype
  • Gene by gene, variants, and coverage are being analyzed on powerful IOBIO servers
  • Click on a gene and visualize the variants, colored by the impact. See coverage levels across the gene
  • As genes are analyzed, filters are pinpointing variants for closer examination. Customize the filters and the variants are instantly reassessed
  • Review each filtered variant to assess pathogenicity and association with known phenotypes

genepanel.iobio
Genepanel.iobio generates comprehensive gene lists, leveraging multiple genetics research community resources; including commercially-available gene panel tests and phenotype term-driven gene list generation tools.

There are three main resources in Genepanel.iobio:

  • Genetic Testing Registry: Allows to compile a gene list for suspected conditions
  • Phenolyzer: Generates a gene list for suspected phenotypes
  • Add Genes: Provides an option to add a suspected genes

Results from all these resources are propagated to the Summary page, which is a comprehensive gene list for the suspected condition and its associated phenotypes, along with the manually added genes.

bam.iobio

Bam.iobio provides 3 primary statistics:

  • Estimated read coverage is shown linearly across the genome
  • Read coverage distribution
  • Fragment length distribution

The dashboard can be zoomed to show statistics for a single chromosome at a time. The app works for both BAM and CRAM format files. Files will normally be processed by providing a URL to the app, but local files may also be used.

vcf.iobio

vcf.iobio shows high level variant calling (VCF) metrics in real time. Including:

  • Variant density - number of variant calls across all autosomes and sex chromosomes
  • Ts/Tv ratio - ratio of transitions to transversions
  • Base changes - A>G/C/T, G>A/C/T, C>A/G/T, T>A/G/C
  • Variant types - SNPs, insertions, deletions
